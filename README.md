Projeto JavaScript com Manipulação de DOM e Eventos
Desenvolver uma aplicação interativa em JavaScript que demonstre domínio
avançado de manipulação de DOM, event listeners e objetos nativos do
JavaScript. 
Projeto idealizado para a disciplina de FRONT END II, onde foi criado um "paint interativo" onde o usuário pode desenhar com o mouse, mudar a cor do traço, o tamanho do pincel, 
inclusive salvar o desenho.

Tema: Paint/Desenho Interativo
Crie uma aplicação de desenho onde o usuário OBRIGATORIAMENTE pode
desenhar com o mouse na tela. A caixa de ferramentas (cores, tamanhos,
borracha) NÃO é obrigatória, mas o desenho com mouse é OBRIGATÓRIO. 

Requisitos Obrigatórios
1. Event Listeners.
2. Objetos Nativos do JavaScript.
3. Manipulação de DOM.
4. Funções.
Independente do tema escolhido, seu projeto DEVE ter:
5. Feedback visual para ações do usuário (animações CSS, mudanças de cor, etc.)
